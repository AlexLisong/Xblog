Xblog

主页文档截图实例

强大、响应式的博客框架

composer create-project lufficc/xblog

  • 专注写作

    支持 GitHub Flavored Markdown 的全部特性。 实时保存、预览、发布，文章的多状态管理。

  • 超快速度

    精心优化每一条数据查询和代码调优，更有可选的 Redis 缓存让你的博客在百毫秒内打开。

  • 功能强大

    图片拖拽、粘贴板上传不在话下，更有文章管理、文件管理和完善的评论系统。

  • 更多自定义

    Markdown评论？多说评论? Disqus评论? 谷歌分析？统统不在话下, 打造你的独一无二的博客。
